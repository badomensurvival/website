# Quickshop

O plugin Quickshop é um simples plugin que permite que jogadores criem lojas de items de forma rápida e fácil

### Como criar uma loja?

Você pode criar uma loja segurando um item na mão, **aganchando** e batendo em um baú. Você pode alterar o item da loja usando o comando **/qs item**.

### Comandos

> **/qs help** - Mostra a ajuda do QuickShop.

> **/qs unlimited** - Dá a uma loja estoque ilimitado.

> **/qs setowner** - Altera a propriedade de uma loja.

> **/qs amount** - Define a quantidade de itens em uma loja (útil quando há problemas com o chat).

> **/qs buy** - Altera uma loja para o modo COMPRA.

> **/qs sell** - Altera uma loja para o modo VENDA.

> **/qs price** - Altera o preço de compra/venda de uma loja.

> **/qs remove** - Remove a loja em que você está olhando.

> **/qs empty** - Remove todos os itens de uma loja.

> **/qs refill** - Adiciona uma determinada quantidade de itens a uma loja.

> **/qs clean** - Remove todas as lojas (carregadas) sem estoque.

> **/qs fetchmessage** - Mostra mensagens não lidas de lojas.

> **/qs create** - Cria uma nova loja a partir do baú selecionado.

> **/qs find** - Localiza a loja mais próxima de um tipo específico.

> **/qs size** - Altera a quantidade por pacote de uma loja.

> **/qs item** - Altera o item da loja.
