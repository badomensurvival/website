# Death Chest :skull:

O plugin Death Chest é responsável por gerenciar os itens dos jogadores após a morte. Este plugin simplesmente gera um baú com os itens do jogador, permitindo que eles os recuperem facilmente, sem se preocupar que alguém os roube ou que se percam. O PRO DeathChest é adequado para qualquer tipo de servidor!

### Recursos Principais
- Geração automática de um baú contendo os itens do jogador após a morte.
- Facilita com que os items não sejam perdidos.
- Facilidade na recuperação dos itens pelos jogadores.

### Comandos

> **/dc list** - Lista todos os baús do jogador que estejam ativos.
Nele é possível visualizar os baús, bem como a data de criação e a localização, além de permitir o teleporte até o baú (**C$ 2.000,00**)
