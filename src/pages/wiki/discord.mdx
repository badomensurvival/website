# Discord

O servidor de minecraft possui um bot no discord que permite que os jogadores interajam com o servidor através do discord.

### Comandos
Para que tais comandos funcionem, é necessário que o jogador tenha sua conta do discord vinculada ao servidor.

> **/inv** - Mostra o inventário do jogador

> **/ender** - Mostra o enderchest do jogador

> **/item \<slot\>** - Mostra o item do slot selecionado

> **/playerinfo \<player\>** - Mostra informações do jogador

> **/playerlist** - Mostra a lista de jogadores online

> **/leaderboard \<leaderboard\>** - Mostra o ranking de jogadores de acordo com o leaderboard selecionado
