# Bem Vindo

Aqui você aprenderá tudo que precisa saber sobre o Bad Omen Hard Survival, desde os comandos, alguns sistemas e muito mais!

O Servidor Bad Omen há algum tempo e tem a intenção de entreter todos seus jogadores, com o objetivo de ser um Servidor focado principalmente na comunidade, este foi o nosso propósito desde o começo do Servidor!

### 🖥️ Java Edition
**Versão 1.19.4**

Como entrar?

- Clique em Multijogador > Adicionar Servidor > Colocar o IP > Concluido.
- IP: badomen.fun Porta: 25565 (Padrão)
- Para se registrar, dê /register suasenha suasenha (precisa repetir 2 vezes a senha)

### 🎮 Bedrock Edition (Celular, Xbox One, PS)

**Versão 1.19.81**

Como Entrar?

- Clique em Jogar > Servidores > Adicionar Servidor > Colocar o IP e a Porta > Salvar.
- IP: badomen.fun Porta: 19132 (Padrão)
- Para se registrar, dê /register suasenha suasenha (precisa repetir 2 vezes a senha)
