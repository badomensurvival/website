# Skins

Skin customizada serve para que o jogador sem Minecraft Original possa usar skins.

#### Adicionar Skin por Link
> Método Recomendado!

Envie sua skin em qualquer site de armazenamento de imagens.

⚠️ Site indicado: https://imgur.com/

1. Copie o link direto da imagem (link com final .png).
2. Dê /skin url link.com/final.png

🎯 Exemplo: `/skin url https://i.imgur.com/BL2ONc9.png`

#### Adicionar Skin por Nick
Procure contas originais com a skin que deseja usar.

⚠️ Site indicado: https://namemc.com/

- /skin set [nick] - Adicionar a mesma skin da conta orginal.

🎯 Exemplo: `/skin set Notch`
