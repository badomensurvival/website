# Ranks

Os ranks são uma forma de classificar os jogadores de acordo com seu avanço no servidor. Para avançar em cada nível é necessário que o jogador complete certos desafios. Após completo um rank são liberadas recompensas e novos desafios.

Atualmente o servidor conta com 4 ranks diferentes (mais serão adicionados em breve), sendo eles:

- <span style={{color:'#AAAAAA'}}>**ROOKIE**</span> (Todos iniciam com este rank)
- <span style={{color:'#00AA00'}}>**CASTAWAY**</span>
- <span style={{color:'#00AA00'}}>**SURVIVOR**</span>
- <span style={{color:'#00AA00'}}>**ADVENTURER**</span>
