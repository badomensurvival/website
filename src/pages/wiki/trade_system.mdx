# Trade System

O plugin Trade System é um plugin de troca de itens entre jogadores, com ele você pode trocar itens com outros jogadores de forma segura e rápida.

### Comandos

> **/trade \<jogador\>** - Inicia uma troca com o jogador especificado.

> **/trade accept** - Aceita a troca.

> **/trade deny** - Recusa a troca.

> **/trade toggle** - Ativa ou desativa as trocas.
